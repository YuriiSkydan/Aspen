# Aspen
My first 2D Game Engine

Game Engine supports Physics, Animations, Audio and C++ Scripting.
Game Engine Components:
Transform,
SpriteRenderer,
BoxCollider,
CircleCollider,
PolygonCollider,
Rigidbody,
Camera,
Animator,
AudioSource,
AudioListener
